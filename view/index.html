FCC: URL Shortener


Example Creation usage:

https://fcc-dc-short-url.herokuapp.com/new/https://www.google.com

Example output:

{ "original_url": "https://www.google.com",
  "short_url": "https://fcc-dc-short-url.herokuapp.com/0" }

Example usage:

Clicking on the following link...

https://fcc-dc-short-url.herokuapp.com/0

...will take you to your original link!

https://www.google.com
